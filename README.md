# RetroNES

RetroNES is a plugin for NES (Nintendo Entertainment System) game emulation, featuring over 400 Chinese-localized classic ROMs. It supports online play, and you can enjoy games through VSCode, Cursor plugin, or directly in your browser.

---

## Online Play

[Play RetroNES Games Online](https://kuangyx.cn/pages/game.html)

---

## Plugin Introduction

- **Support for 400+ NES Game ROMs**: Load and play your favorite NES classics.
- **Full Chinese Localization**: All games are fully localized to Chinese for a better experience.
- **Cross-Platform Support**: Play via VSCode plugin, Cursor plugin, or directly in the browser.

---

## Installation

### Install via VSCode Marketplace

1. Open VSCode.
2. Visit the [VSCode Marketplace](https://marketplace.visualstudio.com/items?itemName=tomiaa.RetroNES).
3. Search for "RetroNES" and click Install.

### Install via Cursor Plugin

1. Open the Cursor plugin marketplace.
2. Search for "RetroNES" and click Install.
